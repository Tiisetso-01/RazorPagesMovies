This project was built as part of Workshop 1 for the ASP.NET Razor Pages tutorial.

## Getting Started

### Prerequisites
- .NET SDK 10.0+
- Visual Studio Code or Antigravity IDE with C# Dev Kit

### Run the application

```bash
dotnet run
```

The app will be available at `https://localhost:5001` or `http://localhost:5000`.
